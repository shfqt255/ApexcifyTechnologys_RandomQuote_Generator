# Random Quote Generator (Flutter)

A simple  Random Quote Generator App built with Flutter.
The app fetches quotes from a public API and displays them with a clean UI and smooth state management using Provider.
This project was developed as part of an internship task assigned by Apexify Technologys.

# Features

- Fetches random quotes from an API
- Displays quote + author
- Floating action button to load a new quote
- Provider state management
- Error handling
- Lightweight and beginner-friendly code structure
